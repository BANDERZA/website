Plan & Design

Goals:

  • Must create device that counts steps of the user
  • Calculates calories burned from steps
  • Notify user if they have reached health goal of (30 min)
  • Log the time, steps walked, and calories counted into a csv file
  • Calculates the estimated amount of steps the user took
  • Displays steps, calories burned and distance on screen for user to see

    • A microbit is a programmable device created by BBC with a built-in USB interface, motion sensors, temperature sensors, buttons and 25 Leds. It can be programmed using the Make code micro bit website. I decided to use the micro bit because it was small and easy to fit around the foot or ankle, also it is very simple to program.

    The Micro Bit measures the acceleration strength and if the strength is above 1500 the code adds 1 to the variable” steps” which is set to zero at the start. The number of calories is also set to zero at the start of the code. Using a forever block the calories are forever set to the number of steps times 0.4. For the distance I simply multiplied the number of steps by 0.74 which is the average amount of meteeres a steps is for a person.

    I also want to notify the user if they have reached the goal of walking for 30 min. Automatically the micro bit counts the amount time (in milliseconds) that has passed when it turns on and resets when you reset entire code. If the time is greater than or equal to 1800000 milliseconds (30 minutes) then the micro bit plays a melody, this will be the notifier to the user. However, there is the problem that the user could just leave the micro bit on for thirty minutes without walking and the melody will still play so in the” if” statement I made it so that the number of steps walked also must be greater than 3000 as that is the number of steps taken in 30 mins on average.

    Every time the number of steps changed it is logged with the calories burned and the time after the start of the program the step was taken. This can be stored later as a csv file and called on in python code.

    I wanted to show the user the number of steps the user has walked and the calories they have burned I decided to use the micro bit built in Leds to show this. So, in a forever loop I made it so that it shows the number of steps and then the number of calories burned .It will cycle between both until the program is finished
